🔐 Google Forms 統合ガイド

Design Guildの応募フォームをセキュアに実装する方法

📋 方法1: カスタムデザイン + Google Forms API

現在の美しいフォームデザインを維持しながら、データをGoogle Formsに送信

✅ メリット

  • デザインの完全なカスタマイズ
  • ユーザー体験の統一性
  • リアルタイムバリデーション
  • アニメーション効果の維持

⚠️ デメリット

  • 実装がやや複雑
  • Google Forms IDの取得が必要
  • CORSの設定が必要な場合あり

実装手順

1 Google Formsを作成

Google Formsで応募フォームを作成し、各フィールドのname属性を取得

2 フォームIDを取得

URLから https://docs.google.com/forms/d/e/[FORM_ID]/viewform のFORM_IDを取得

3 JavaScriptを実装 
// Google Formsへの送信
async function submitToGoogleForms(formData) {
    const GOOGLE_FORM_ID = 'YOUR_FORM_ID_HERE';
    const url = `https://docs.google.com/forms/d/e/${GOOGLE_FORM_ID}/formResponse`;
    
    // フィールドマッピング（Google Formsの入力フィールドのname属性）
    const params = new URLSearchParams({
        'entry.123456789': formData.name,      // お名前
        'entry.987654321': formData.email,     // メール
        'entry.456789123': formData.profession, // 職種
        'entry.789123456': formData.motivation  // 動機
    });
    
    try {
        // iframe経由で送信（CORS回避）
        const iframe = document.createElement('iframe');
        iframe.style.display = 'none';
        iframe.src = `${url}?${params.toString()}&submit=Submit`;
        document.body.appendChild(iframe);
        
        // 成功メッセージ表示
        setTimeout(() => {
            document.body.removeChild(iframe);
            showSuccessMessage();
        }, 2000);
        
        return true;
    } catch (error) {
        console.error('送信エラー:', error);
        return false;
    }
}

🖼️ 方法2: Google Forms iframe埋め込み

Google Formsを直接ページに埋め込む最もシンプルな方法

✅ メリット

  • 実装が超簡単（5分で完了）
  • メンテナンス不要
  • 確実に動作
  • 自動でレスポンシブ対応

⚠️ デメリット

  • デザインのカスタマイズに制限
  • Googleブランディング表示
  • サイトデザインとの統一感が難しい

実装手順

1 Google Formsで埋め込みコードを取得

フォーム編集画面 → 送信 → <>アイコン → HTMLコードをコピー

2 HTMLに埋め込み 
<!-- モーダル内のフォームを置き換え -->
<div class="google-form-container">
    <iframe 
        src="https://docs.google.com/forms/d/e/YOUR_FORM_ID/viewform?embedded=true"
        width="100%" 
        height="800" 
        frameborder="0" 
        marginheight="0" 
        marginwidth="0">
        読み込んでいます…
    </iframe>
</div>
3 スタイリング調整 
.google-form-container {
    background: white;
    border-radius: 12px;
    padding: 1rem;
    box-shadow: 0 10px 30px rgba(0,0,0,0.1);
}

.google-form-container iframe {
    border-radius: 8px;
    min-height: 600px;
}

🎯 推奨実装方法

GitHub Pagesで公開する場合：方法2（iframe埋め込み）

実装が簡単で、確実に動作し、個人情報も完全に保護されます。
デザインをある程度犠牲にしても、セキュリティと実装の簡単さを優先することをお勧めします。

🛠️ Google Forms作成時の推奨設定

  1. 必須項目の設定：名前、メール、職種、動機を必須に
  2. メール収集：設定 → レスポンス → メールアドレスを収集
  3. 確認メール：回答のコピーを回答者に送信
  4. 通知設定：新しい回答についてメール通知を受け取る
  5. スプレッドシート連携：回答タブ → スプレッドシートアイコン

📝 実装後のテスト項目

  • ✅ フォーム送信が正常に完了するか
  • ✅ Google スプレッドシートにデータが記録されるか
  • ✅ 確認メールが送信されるか
  • ✅ 管理者に通知が届くか
  • ✅ モバイル表示が適切か
  • ✅ エラー時の挙動が適切か
