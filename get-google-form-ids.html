🔧 Google Forms エントリーID取得ガイド

⚠️ 重要: Google FormsのエントリーIDが正しくないと、フォーム送信が失敗します。

ステップ 1: Google Formsを開く

1. Google Formsにアクセス

2. Design Guild用のフォームを開く（または新規作成）

ステップ 2: 事前入力リンクを取得

1. フォーム編集画面の右上の「⋮」メニューをクリック

2. 「事前入力されたリンクを取得」を選択

3. 各フィールドにテストデータを入力：

  • お名前: テスト太郎
  • メール: test@example.com
  • 職種: Webデザイナー
  • 経験年数: 1-3年
  • 動機: テストメッセージ

4. 「リンクを取得」をクリック

ステップ 3: URLからエントリーIDを抽出

生成されたURLは次のような形式になります：

https://docs.google.com/forms/d/e/1FAIpQLSe.../viewform? entry.123456789=テスト太郎& entry.987654321=test@example.com& entry.456789123=Webデザイナー& entry.789123456=1-3年& entry.111222333=テストメッセージ

各entry.XXXXXXXXXの数字がエントリーIDです。

ステップ 4: コードを更新

main.jsの以下の部分を更新してください：

// エントリーIDのマッピング（Google Formsのフィールドに対応） const params = new URLSearchParams({ 'entry.XXXXXXXXX': data.name, // お名前のID 'entry.XXXXXXXXX': data.email, // メールアドレスのID 'entry.XXXXXXXXX': data.profession, // 職種のID 'entry.XXXXXXXXX': data.experience || '', // 経験年数のID 'entry.XXXXXXXXX': data.motivation, // 応募動機のID 'entry.XXXXXXXXX': data.portfolio || '' // ポートフォリオURLのID });

デバッグ用テストコード

以下のボタンをクリックして、ブラウザコンソールで動作を確認できます：

function testGoogleForms() { console.log('=== Google Forms テスト送信 ==='); // テストデータ const testData = { name: 'テスト太郎', email: 'test@example.com', profession: 'Webデザイナー', experience: '1-3年', motivation: 'テスト送信です' }; // フォームIDとURL const FORM_ID = '1FAIpQLSe8b_ynVU1_TqQuoV472_eVFScWgj2WWaeRWFZDmKjkIKQi7Q'; const url = `https://docs.google.com/forms/d/e/${FORM_ID}/formResponse`; console.log('送信URL:', url); console.log('送信データ:', testData); // エントリーIDマッピング const params = new URLSearchParams({ 'entry.1751089080': testData.name, 'entry.2019842798': testData.email, 'entry.61724704': testData.profession, 'entry.1887838426': testData.experience, 'entry.505500388': testData.motivation }); console.log('パラメータ:', params.toString()); // テスト用iframe作成 const iframe = document.createElement('iframe'); iframe.style.display = 'none'; iframe.name = 'test_iframe'; document.body.appendChild(iframe); // フォーム作成 const form = document.createElement('form'); form.method = 'POST'; form.action = url; form.target = 'test_iframe'; params.forEach((value, key) => { const input = document.createElement('input'); input.type = 'hidden'; input.name = key; input.value = value; form.appendChild(input); }); document.body.appendChild(form); console.log('送信開始...'); form.submit(); setTimeout(() => { document.body.removeChild(iframe); document.body.removeChild(form); console.log('送信完了（確認はGoogle Formsの回答タブで）'); alert('テスト送信完了！Google Formsの回答タブを確認してください。'); }, 2000); }

代替案: mailto方式（確実に動作）

Google Formsがうまく動作しない場合は、mailto方式に切り替えることができます：

// main.jsのsubmitToGoogleFormsを以下に置き換え async function submitToGoogleForms(event) { event.preventDefault(); const form = event.target; const formData = new FormData(form); const data = { name: formData.get('name'), email: formData.get('email'), profession: formData.get('profession') === 'other' ? formData.get('otherProfession') : formData.get('profession'), experience: formData.get('experience'), motivation: formData.get('motivation'), portfolio: formData.get('portfolio') }; // メール本文を作成 const subject = 'Design Guild 参加申込み'; const body = ` Design Guild 参加申込み 【お名前】${data.name} 【メールアドレス】${data.email} 【職種・専門分野】${data.profession} 【経験年数】${data.experience || '未回答'} 【参加動機・期待すること】 ${data.motivation} 【ポートフォリオURL】${data.portfolio || 'なし'} ※このメールは自動生成されています。 `.trim(); // mailtoリンクを開く const mailtoLink = `mailto:hello@design-guild.jp?subject=${encodeURIComponent(subject)}&body=${encodeURIComponent(body)}`; window.location.href = mailtoLink; // 成功メッセージを表示 setTimeout(() => { form.style.display = 'none'; const successDiv = document.getElementById('formSuccess'); if (successDiv) { successDiv.style.display = 'block'; } }, 1000); }
