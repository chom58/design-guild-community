1現在のイベントデータをコピー

データを読み込みました！下のテキストエリアで編集できます。

2イベントを追加・編集

テキストエリアで直接編集してください。新しいイベントを追加する場合は、以下のテンプレートをコピーして使ってください。

📝 イベントテンプレート（クリックで開く）
{ "id": "event-20240901", "title": "イベント名をここに入力", "type": "meetup", "typeLabel": "交流会", "date": "2024-09-01", "time": "15:00-17:00", "location": "場所をここに入力", "capacity": 20, "currentParticipants": 0, "description": "イベントの説明をここに入力", "features": [ "特徴1", "特徴2", "特徴3" ], "image": "", "registrationOpen": true, "registrationDeadline": "2024-08-30", "mailSubject": "【参加申込】イベント名（日付）", "mailBody": "申込メールの本文テンプレート", "tags": ["タグ1", "タグ2"] }
💡 ヒント: カンマ「,」を忘れずに！最後のイベントの後ろにはカンマ不要
